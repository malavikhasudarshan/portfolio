Catch the Coo!

Live Demo
Code

"Catch the Coo!" is a whimsical game I developed using p5.js, HTML and JavaScript, inspired by the picturesque Scottish hills I call home. Players take on the role of Scott, an adventurous sheep, tasked with helping his friend Hamish—a highland cow with overgrown hair and limited visibility—find his way home. Players explore the vibrant, handrawn landscape using simple controls: arrow keys for movement and the space bar to jump. See what happens if you try to move Scott off the edge of the screen!

Features

The game was designed to be approachable for all ages, incorporating:
  • Collision Detection: A bounding box system ensures accurate interactions between Scott and Hamish.
  • Dynamic Environments: The background changes color as Scott explores.
  • Custom Artwork: I created the visuals using Adobe Fresco for the sheep and scenery and Pixilart for Hamish, blending hand-drawn and pixel art styles.
